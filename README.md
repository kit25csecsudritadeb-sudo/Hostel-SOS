# HostelSOS

HostelSOS is a hostel issue-reporting application built for the Google AdMob App Development Challenge.  
It allows students to anonymously report maintenance, safety, and cleanliness issues while enabling hostel management to track and resolve complaints efficiently.

## Features
- Anonymous issue reporting  
- Photo evidence upload  
- Severity tagging  
- Issue status tracking (Pending, In Progress, Resolved)  
- Analytics dashboard  
- Notifications  
- AdMob monetization

## Technology
Built using MIT App Inventor.  
The `.aia` source file is included in this repository.
